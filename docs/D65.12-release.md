## 1. Executive Summary

**D65 — Window Sessions Foundation** introduce el concepto de **Session** como unidad persistible del workspace: infraestructura desacoplada, serializable, sin UI de producto y sin I/O. La serie queda **RELEASED** y **CLOSED**.
