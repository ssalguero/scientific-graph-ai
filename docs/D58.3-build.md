---

## 1. Arquitectura implementada

```text
Pointer → ResizeHandle → Resize Math (computeResizedGeometry)
  → GeometryConstraints (min/max + W/N re-anchor)
  → WorkspaceConstraints (edge-aware clamp)
  → WindowGeometryState.set (immutable)
  → FloatingWindowBridge → Layer → FloatingWindow
```

---

## 2. Pipeline de constraints

| Paso | Función | Contrato |
|------|---------|----------|
| 1 | `computeResizedGeometry` | Edge math from `WindowResizeOrigin` |
| 2 | `applyGeometryConstraints` | `{ minWidth, minHeight, maxWidth?, maxHeight? }` |
| 3 | `applyWorkspaceConstraints` | `{ left, top, right, bottom }` |
| 4 | `geometryState.set` | SSOT write |

Defaults (hasta Layout Engine dinámico):

- Geometry: `minWidth: 160`, `minHeight: 100`
- Workspace: `0,0`–`1920,1080`
