## 7. Inventario — Sidebar inline

### 7.1 Ubicación y estructura

| Elemento | Ubicación |
|----------|-----------|
| JSX `<aside>` | `page.tsx` **L19737–L20056** (320 LOC) |
| Helper `DashboardSection` | `page.tsx` **L504–L547** |
| State relevante | p.ej. `sidebarGraphLibraryOpen` (~L15310), flags de paneles |

Estructura lógica del aside (orden de render):

1. Header — «📊 Dashboard Científico»
2. Grupo **Curvas matemáticas (NUBE)** — nueva/vaciar + biblioteca colapsable
3. Divider
4. Grupo **Proyecto científico (.SGPROJ)** — `ProjectScientificFilePanel`, Actividad (`HistoryPanel`), `LocalProjectsPanel`, Restablecer
5. Divider
6. `DashboardSection` **Módulos** (🧩) — `ScientificModuleCard`
7. `DashboardSection` **Herramientas** (🧠) — Asistente / Reportes (module-gated)
8. `DashboardSection` **Recursos** (📚) — Biblioteca / Historial (`RecentProjectsPanel`)
9. `DashboardSection` **Sistema** (⚙) — Configuración (`SettingsPanel`)

### 7.2 Navegación (comportamiento actual — a preservar)

No hay router declarativo. Los ítems usan `onClick` + state:

- `setActiveWorkspaceSection(...)`
- `setAnalysisInspectorSection("advisor")`
- `setControlPanelTab("library")`
- Flags locales: `sidebarGraphLibraryOpen`, `projectActivityPanelOpen`, `recentProjectsPanelOpen`, `settingsPanelOpen`

### 7.3 Class tokens del sidebar

Ver §4.4. Shell del `<aside>`:  
`w-full lg:w-[260px] lg:max-w-[280px] xl:w-[280px] xl:max-w-[280px] … border-r …` (string inline en JSX, no constante nombrada).

### 7.4 Props públicas futuras (contrato documental D45)

Para D45.4, el framework `Sidebar*` deberá aceptar (como mínimo, sin cambio visual):

| Superficie | Props conceptuales (freeze documental) |
|------------|----------------------------------------|
| `Sidebar` | `children`, `className?` — shell aside idéntico |
| `SidebarSection` | `title`, `icon`, `defaultOpen?`, `children` (equiv. `DashboardSection`) |
| `SidebarGroup` | agrupación + label opcional |
| `SidebarItem` | label, onClick, disabled/aria, badge?, expanded caret? |
| `SidebarFooter` | slot inferior opcional |

**Freeze:** estos nombres/contratos se documentan aquí; la implementación ocurre en D45.3/D45.4. No se crean archivos en D45.1.
