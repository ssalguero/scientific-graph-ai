## 1. Pipeline unificado

```text
Pointer
  ↓
Resize Math
  ↓
GeometryConstraints
  ↓
WorkspaceConstraints
  ↓
SnapTargetProviders.collect (Workspace + Window)
  ↓
applyMagneticSnap()
  ↓
acceptSnapAxes (discard axis if violates constraints / workspace)
  ↓
WindowGeometryState.set
```

```text
Drag (D59.2):   Math → compose(Snap) → GeometryState
Resize (D59.3): Math → GeometryConstraints → WorkspaceConstraints → compose(Snap) → GeometryState

Same Engine · Same Providers · Decoupled from both bridges
```

```text

```
