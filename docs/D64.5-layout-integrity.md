## 2. Árbol arquitectónico oficial (verificado)

```text
UI (lib/ui + ui chrome)
  → Workspace (shell)
      → Layout Engine          (SSOT espacial · sole consumer = WorkspaceLayout)
      → panels slot
          → Docking            (page compone DockRoot en el slot)
          → FloatingWindowBridge → FloatingWindowLayer
  → WindowManager              (page · public windows barrel)
  → Series / Tabs / Tab-UI / Content   (library-only · no page mount)
```

**Composición en producto (page):** `WindowManager` → `WorkspaceLayout` (sidebar / content / panels) · `DockRoot` dentro de panels · sin import directo de `layout-engine` · sin series/tabs/content.
