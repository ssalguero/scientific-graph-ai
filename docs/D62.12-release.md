## 2. Objetivo de la serie

Entregar la capa operable sobre Window Tabs Foundation (D61):

- Mutador público aditivo `TabRegistry.setState`
- Selection Policy externa (`afterUnregister` / `ensureActive`)
- Document Switch React-agnóstico (`TabId → Opaque Content Handle`)
- Chrome presentacional `tab-ui/` (`TabStrip` / `TabBar` / `TabDocumentHost`)
- Barrels públicos + validators

**Sin** Tab↔Series wiring, persistencia, drag-reorder ni population obligatoria de producto.

---
