---

## 1. Arquitectura implementada

```text
Pointer
  ↓
FloatingWindowResizeHandle (n|s|e|w|ne|nw|se|sw)
  ↓
WindowResizeBridge (begin / update / end + edge math)
  ↓
WindowGeometryState (immutable set)
  ↓
FloatingWindowBridge → FloatingWindowLayer → FloatingWindow
```

**Drag XOR Resize:** `WindowManager` wrappers — `beginDrag` llama `endResize`; `beginResize` llama `endDrag`.

---

## 2. Flujo Pointer → ResizeBridge

```text
pointerdown on handle
  → stopPropagation
  → setPointerCapture
  → beginResize(id, edge, clientX, clientY)  // freezes WindowResizeOrigin
pointermove
  → updateResize(clientX, clientY)           // computeResizedGeometry → GeometryState.set
pointerup
  → endResize()
  → releasePointerCapture
```
