## 2. Arquitectura final certificada

### Interaction pipeline

```text
Pointer
  ↓
ResizeHandle (n|s|e|w|ne|nw|se|sw)
  ↓
ResizeBridge (begin / update / end)
  ↓
GeometryState
  ↓
FloatingBridge
  ↓
FloatingLayer
  ↓
FloatingWindow
```

### Constraint pipeline

```text
Resize Math (computeResizedGeometry)
  ↓
Geometry Constraints (min/max + W/N re-anchor)
  ↓
Workspace Constraints (edge-aware clamp)
  ↓
GeometryState
```

### Host stack

```text
WindowManager
 ├── WindowState / WindowAPI          (D55 FROZEN)
 ├── WindowGeometryState              (geometry SSOT)
 ├── WindowDragBridge                 (sole move path)
 ├── WindowResizeBridge               (sole resize path)
 ├── WindowGeometryProvider
 ├── WindowDragProvider
 └── WindowResizeProvider
              │
FloatingWindowBridge → FloatingWindowLayer → FloatingWindow
              (+ FloatingWindowResizeHandle x8)
```

### Dual interaction exclusivity

| Regla | Estado |
|-------|--------|
| Drag XOR Resize | **CERTIFIED** |
| `beginDrag` ends resize | **CONFIRMED** |
| `beginResize` ends drag | **CONFIRMED** |

## 8. Decisiones congeladas

| Decisión | Valor |
|----------|-------|
| Geometry authority | `WindowGeometryState` (storage-agnostic concept) |
| Resize Origin | `{ pointerX, pointerY, geometry }` |
| Resize API | `beginResize` / `updateResize` / `endResize` only |
| Constraints split | GeometryConstraints vs WorkspaceConstraints |
| Coordinate Space | Absolute layer coords; no viewport/scroll/zoom/DPR |
| Cancel semantics | `endResize` + session replace (no `cancelResize` on API) |
| Interaction exclusivity | Drag XOR Resize |

---

## 9. Riesgos cerrados

| Riesgo | Mitigación certificada |
|--------|------------------------|
| Resize + drag concurrent | XOR en WindowManager |
| N/W edges + size | GeometryState + Origin re-anchor |
| Mutación de objetos | `geometry-immutable-input` / `cloneGeometry` |
| Viewport / DPI creep | Coordinate Space Freeze + boundaries validator |
| Barrel leaks | `validate-d58-resize-api` |
