## 3. validate-d53-api-freeze

**Propósito:** Garantizar que el contrato congelado en [`D53.1-api-freeze.md`](D53.1-api-freeze.md) no se rompe.

### Checks previstos

| ID | Verificación |
|----|--------------|
| `d53.api.providerExists` | `DockInteractionProvider` existe |
| `d53.api.contextExists` | `DockInteractionContext` / context de interacción existe |
| `d53.api.hooksExist` | `useDockInteraction` · `useDockDrag` · `useDockResize` existen |
| `d53.api.stateFields` | State incluye: `focusedDock` · `activeDock` · `hoverDock` · `draggingDock` · `resizingDock` · `dragSession` · `resizeSession` |
| `d53.api.dragSessionShape` | `DockDragSession` tiene `dockId` · `pointerId` · `start` · `current` |
| `d53.api.resizeSessionShape` | `DockResizeSession` tiene `dockId` · `pointerId` · `edge` · `start` · `current` |
| `d53.api.resizeEdge` | `ResizeEdge` union congelada presente |
| `d53.api.methodsFocus` | `focus` · `blur` · `activate` · `deactivate` presentes |
| `d53.api.methodsDrag` | `beginDrag` · `updateDrag` · `endDrag` presentes |
| `d53.api.methodsResize` | `beginResize` · `updateResize` · `endResize` presentes |
| `d53.api.barrelAdditive` | Barrel exporta superficie interaction de forma aditiva; no elimina exports D51/D52 |
| `d53.api.dockStateUntouched` | `DockState` sigue siendo solo `activePanelIds` + `sizes` |

```text
STUBS OK — presence of methods required; rich bodies not required
```

## 4. validate-d53-interactions

**Propósito:** Verificar que la infraestructura de interacción está cableada como capa, sin comportamiento avanzado D54+.

### Checks previstos

|----|--------------|
| `d53.ix.providerNested` | `DockRoot` anida `DockInteractionProvider` bajo `DockProvider` |

| `d53.ix.singleActive` | Documentado/enforced: un solo `activeDock` (checklist + shape) |

## 5. validate-d53-governance

**Propósito:** Aislar interaction de ciencia, de ciclos, y de Zero UX violations.

### Checks previstos

| ID | Verificación |
|----|--------------|
| `d53.gov.noScientificImports` | Módulos docking/interaction sin imports graphs/math/statistics/analysis/export/supabase |
| `d53.gov.noTailwindNew` | Sin clases Tailwind nuevas en archivos interaction |
| `d53.gov.noHardcodedColors` | Sin hex/rgb hardcodeados nuevos en interaction |
| `d53.gov.noPageInteractionImports` | `page.tsx` no importa API de interaction |
| `d53.gov.contextIsolated` | Interaction state no mergeado en `DockState` |
| `d53.gov.oneWayNesting` | **`DockInteractionProvider` never imports `DockProvider`** |
| `d53.gov.noReverseImport` | `DockContext` / `DockProvider` no importan módulos interaction |
| `d53.gov.noCircular` | Sin dependencia circular Provider ↔ Interaction |

### Check crítico (one-way)

```text
ALLOWED:
  DockProvider → DockInteractionProvider

FORBIDDEN:
  DockInteractionProvider imports DockProvider
  DockProvider imports DockInteractionProvider
```

Consumo permitido hacia abajo: `useDockContext` desde hijos bajo ambos providers. **Nunca** crear `DockProvider` desde interaction.

| # | Criterio | Evidencia esperada |
|---|----------|--------------------|

| 10 | **Zero UX** | Sin CSS/animaciones nuevas · sin cambios funcionales científicos · sin wiring visual en `page.tsx` |
