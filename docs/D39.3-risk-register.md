## 1. Objetivo

Definir el **Risk Register oficial de ejecución** de PROD-3, heredando íntegramente la evaluación de riesgos de D37.5 y estableciendo un mecanismo operativo de seguimiento durante las futuras épicas.

| Afirmación | Valor |
|------------|-------|
| Reevalúa riesgos / modifica puntuaciones | **No** |
| Altera prioridades / orden de épicas | **No** |
| Crea restricciones arquitectónicas nuevas | **No** |
| Cierra riesgos por aprobación de este documento | **No** |
| Autoriza BUILD | **No** |

---

## 2. Principios

```text
• D37.5 permanece como evaluación oficial de riesgos.
• D39.3 únicamente registra y operacionaliza dichos riesgos.
• Ningún riesgo se considera cerrado por la mera aprobación de este documento.
• Probabilidad, impacto y descripción esencial de IDs R-* / DOC-P3-* se heredan sin re-score.
```

---

## 3. Regla de mantenimiento

```text
Los riesgos podrán cambiar de estado (abierto, mitigado, cerrado o archivado)
durante la ejecución de las épicas, pero no podrán eliminarse del registro histórico.
```

Estados permitidos: **Abierto** · **Mitigado** · **Cerrado** · **Archivado** · **Aceptado** (solo para riesgos documentales ya aceptados en D37.5 §6 — sin eliminar la fila).
