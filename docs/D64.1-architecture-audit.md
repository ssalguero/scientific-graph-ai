## 3. Auditoría de dependencias

### 3.1 Grafo permitido (observado)

```text
lib/ui  ←  ui / toolbar / inspector / docking / workspace (tokens|theme)
layout-engine  ←  workspace (WorkspaceLayout only)
windows (public barrel)  ←  workspace (FloatingWindowBridge only)
tabs barrel  ←  tab-ui · content (types / OpaqueContentHandle)
series barrel  ←  content (TabSeriesBridge mapping only)
```

### 3.2 Ciclos de importación

| Par | ¿Ciclo? | Evidencia |
|-----|---------|-----------|
| workspace ↔ layout-engine | **No** | Solo `workspace → layout-engine` |
| workspace ↔ windows | **No** | Solo `workspace → windows` (FloatingWindowBridge) |
| windows ↔ docking | **No** | Sin imports cruzados |
| tabs ↔ content | **No** | Solo `content → tabs` (type `OpaqueContentHandle`) |
| series ↔ content | **No** | Solo `content → series` (TabSeriesBridge) |
| tabs ↔ series | **No** | Sin edge directo (mapeo vía content) |
| tab-ui ↔ tabs | **No** | Solo `tab-ui → tabs` |
| layout-engine → *externo* | **No** | Solo imports internos del paquete |

**Hallazgo A-DEP-01 — Package import cycles:** **PASS**

### 3.3 Dependencias ocultas / acoplamientos

| Check | Resultado | Clasificación |
|-------|-----------|---------------|
| `windows/**` → `@/lib/scientific` | Ausente | **PASS** |
| `windows/content/**` → `workspace/` | Ausente | **PASS** |
| `windows/content/**` → `WindowManager` / docking / layout-engine | Ausente | **PASS** |
| `series/` → `tabs/` \| `content/` \| `tab-ui/` | Ausente | **PASS** |
| `tabs/` → `series/` \| `content/` \| `tab-ui/` | Ausente | **PASS** |
| `page.tsx` → `windows/series\|tabs\|tab-ui\|content` | Ausente | **PASS** |

| `workspace → windows` (FloatingWindowBridge) | Presente · autorizado D56.4 | **NOTE** (A-DEP-02) |
| `content → tabs` / `content → series` | Presente · autorizado D63 | **NOTE** (A-DEP-03) |
